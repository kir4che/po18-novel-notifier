# PO18 追更小幫手

自動偵測 [PO18](https://www.po18.tw/) 小說更新，發送桌面通知的擴充功能。

## 功能

- 定時檢查追蹤小說有無新章節（預設每 2 小時）
- 新章節桌面通知（點擊跳轉）
- 書籤選單右鍵「加入 PO18 書籍追蹤」
- 手動立即檢查
- 未讀標記 + Badge 計數
- 登入狀態偵測，過期引導重新登入。
- 自訂檢查間隔

## 使用

1. 先登入 PO18（需要登入才能抓取頁面）
2. 在 PO18 書籍頁面按右鍵 >「加入 PO18 書籍追蹤」，或手動貼上網址。
3. 擴充功能會定時檢查更新，有新章節時跳出通知。
4. 點擊通知或擴充功能圖示可檢視追蹤清單
